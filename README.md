# CLI ChatGPT

A simple command-line chatbot that uses OpenAI's GPT-4 model for interactive conversations.

## Requirements

- Node.js >= 18
- OpenAI API key

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set your OpenAI API key:
   ```bash
   export OPENAI_API_KEY=your_api_key
   ```

3. Start the chat:
   ```bash
   npm run chat
   ```

## Usage

- Type your message and press Enter to send
- Type `exit` to quit the chat
- The chat maintains context of the conversation
- Responses are streamed in real-time
